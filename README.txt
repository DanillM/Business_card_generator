# Business Card Generator

## Description

The Business Card Generator is a web application that allows users to create and customize their own digital business cards. Users can input their name, job title, phone number, and email address, and see the changes reflected in real-time on the business card display. The application also provides options to change the card's color and background image.

## Features

- Input fields for name, job title, phone number, and email
- Real-time updates to the business card
- Options to change the card's color (Blue and Yellow)
- Ability to set custom background images
- Responsive design that adjusts with window resizing

## Technologies Used

- HTML
- CSS
- JavaScript
